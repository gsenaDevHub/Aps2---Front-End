# Navegação do Projeto HTML/CSS

Este projeto contém várias páginas HTML organizadas em um menu principal e subpastas de exercícios.

## Como navegar

1. Abra `index.html` na raiz do projeto.
2. O `index.html` é o menu principal.
3. Use os links do menu para acessar as seções:
   - `projetos/aps/Menu.html` — menu dos exercícios de APS
   - `projetos/projeto-inicio/index.html` — página inicial do projeto
   - `projetos/projeto-js/index.html` — projeto JavaScript
   - `projetos/ex-aulas/layout/index.html` — exercício de layout
   - `projetos/ex-aulas/ex01/Untitled-1.html` — formulário do ex01
   - `projetos/ex-aulas/ex02/index.html` — tabela do ex02
   - `projetos/ex-aulas/ex3/new 1.html` — página de links 01 do ex3
   - `projetos/ex-aulas/ex3/new 2.html` — página de links 02 do ex3

## Voltar ao menu principal

As páginas de `projetos/ex-aulas` incluem um link de retorno para o menu principal:

- `../../../index.html`

Isso permite voltar para a página `index.html` a partir das páginas em `projetos/ex-aulas`.
